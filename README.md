Purplle Store Intelligence

AI-powered retail analytics platform that converts CCTV footage into actionable business insights using computer vision and real-time analytics.

Features

* Real-time customer detection and tracking
* Visitor counting and footfall analysis
* Queue monitoring and abandonment detection
* Zone-based heatmap generation
* Conversion funnel analytics
* Interactive dashboard and REST API

Tech Stack

* Python
* YOLOv8
* OpenCV
* ByteTrack
* FastAPI
* Docker

Project Description

Purplle Store Intelligence leverages YOLOv8-based object detection and tracking to analyze customer movement and behavior within retail stores. The platform generates real-time insights such as visitor counts, queue metrics, zone engagement, and conversion analytics, helping businesses optimize store operations and enhance customer experience.
